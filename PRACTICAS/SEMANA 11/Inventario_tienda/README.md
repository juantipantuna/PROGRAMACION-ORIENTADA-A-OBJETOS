# Sistema de Gestión de Inventario

## Descripción
Sistema desarrollado en Python para la gestión de inventario de una tienda, aplicando conceptos de Programación Orientada a Objetos y manejo de colecciones.

## Objetivos
- Aplicar POO en la estructura del programa
- Utilizar colecciones (listas, diccionarios, conjuntos, tuplas)
- Implementar persistencia con archivos JSON
